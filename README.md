# Laravel PostgreSQL Schemas

Laravel package to add support for PostgreSQL `SCHEMA`.

# Installation

```bash
$ composer require bearcodi/laravel-postgresql-schemas
```

## Publish configuration

```bash
$ php artisan config:publish --vendor=bearcodi
```
